# Smart Task Manager

A modern and responsive Task Manager web application built using HTML, CSS, and JavaScript.

## ✨ Features

- ➕ Add tasks
- 🔍 Search tasks
- 🏷️ Task priority
- 📅 Due date support
- ✅ Mark tasks as completed
- 📂 Filter tasks (All, Completed, Pending)
- 🌙 Dark/Light Mode
- 🗑️ Clear all tasks
- 📊 Progress tracker

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
